# ncspot-theme-maker

~ a pretty simple web-based theme maker for [ncspot](https://github.com/hrkfdn/ncspot)!

pick colours, preview them live, and copy the generated theme config straight into your `config.toml`.

## Status

~ currently in development!! planned features:
- [ ] live preview
- [ ] randomize theme colors
- [ ] preset themes (Dracula, Catppuccin, Nord, etc~)
- [ ] save and load themes locally
- [ ] undo/reset changes
- [ ] probably more awesome things if i can think of them
